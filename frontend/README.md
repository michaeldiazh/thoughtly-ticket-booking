# Frontend Application

React + TypeScript frontend for the Thoughtly Ticket Booking System.

## Setup

This frontend directory will contain the React application. To initialize:

```bash
# Using Vite (recommended)
npm create vite@latest . -- --template react-ts

# Or using Create React App
npx create-react-app . --template typescript
```

## Development

Once initialized, run the development server:

```bash
npm run dev
# or
npm start
```

## API Integration

The frontend will connect to the backend API at `http://localhost:3000/api/v1`.
